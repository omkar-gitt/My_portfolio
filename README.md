🌐 Koraganji Omkar Sai — Personal Portfolio

A fully animated, single-page personal portfolio website built with pure HTML, CSS, and JavaScript — no frameworks, no libraries.
✨ Features
🎨 Design & Theme

Dark background with deep indigo, hot pink, cyan, and amber accent colors
Google Fonts: Syne for headings + DM Sans for body text
Noise texture overlay for depth and atmosphere
Custom styled scrollbar with gradient colors

⚡ Animations

Loading screen — Full-screen loader with initials "KOS" and animated gradient progress bar, fades out after 2 seconds
Custom cursor — Glowing cyan dot + trailing indigo ring that follows the mouse
Particle network canvas — 100 floating dots with connecting lines in multiple accent colors
Hero name shimmer — Gradient text animation on the display name
3D floating card — Continuous float animation with spinning dashed rings and glowing orb
Scroll reveal — Fade up, slide left, slide right via IntersectionObserver on all sections
Skill bars — Animate only when scrolled into view, with shimmer sweep after filling
Infinite marquee — Scrolling strip of technologies and companies
Project cards — Top-border gradient sweep on hover + lift effect
Pulsing ripple rings — Three expanding rings in the contact section
Frosted glass navbar — Slides down on page load with backdrop blur

🚀 Getting Started
View Locally

Clone this repository:

bash   git clone https://github.com/omkar-gitt/my-portfolio.git

Open index.html in any modern browser — no build step needed.

Deploy on GitHub Pages

Push this repository to GitHub
Go to Settings → Pages
Set source to main branch
Your site will be live at https://omkar-gitt.github.io/my-portfolio/
